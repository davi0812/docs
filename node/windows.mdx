### Prerequisites
Ensure [Ollama](https://ollama.com/) is installed and running locally. You can download it from [here](https://ollama.com/download).

---

### Step 1: Download the Infera Installer

1. Go to the official Infera GitHub repository:
   - [Infera GitHub Repository](https://github.com/inferanetwork/install-scripts)
   
2. Download the repository by clicking on the green **Code** button and selecting **Download ZIP**.

---

### Step 2: Extract the Folder

1. Once the download is complete, locate the ZIP file in your Downloads folder.
2. Right-click on the ZIP file and select **Extract All** to extract the folder to your preferred location.

---

### Step 3: Move Infera.exe to Desktop

1. Open the extracted folder.
2. Find the file named `Infera.exe`.
3. Drag and drop the `Infera.exe` file onto your desktop for easy access.

---

### Step 4: Ensure Ollama is Running

- **Important**: Make sure Ollama is running on your system. The Infera node will not boot up without Ollama. If Ollama is not already installed, please refer to the [Ollama](https://ollama.com) and ensure it is up and running before proceeding.

---

### Step 5: Boot Up the Infera Node

1. On your desktop, double-click the `Infera.exe` icon.
2. The Infera node will now boot up. If Ollama is not running, you will receive an error and the node will not start.
